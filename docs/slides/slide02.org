** 基本的なデータ構造
   - 下記は基本的なもので標準環境で利用できる
     - ベクトル (vector)
     - 行列 (matrix)
     - リスト (list) 
     - データフレーム (data frame)
     - 配列 (array) (今回は扱わない)
   - /[[color:gray][パッケージなどで拡張することができる]]/

** ベクトルとは
   - スカラー値(単一の値)の集合 
   - スカラー値として扱われる主なもの
     - 数値 (実数や複素数)
     - 文字列 ('や"で囲まれた文字．"foo"，"bar"など)
     - 論理値 ( ~TRUE~ , ~FALSE~ )
   - Rオブジェクトの多くはベクトルとして扱われる\\
     /[[color:gray][スカラーは長さ1のベクトルとして扱われる]]/

** ベクトルの生成と操作
   - 基本的には関数 ~c()~ を用いて作成する
   - 数値や文字列の要素からなるベクトルの生成:
     #+begin_src R :tangle yes
       (x <- c("Alice","Bob","Cathy","David")) # 文字列のベクトル
       (y <- c(1,-2,3,-4,5)) # 数値のベクトル
       (z <- c("apple","berry","cat","dog","elephant")) # 文字列のベクトル
       ## 外側の () は代入した結果の表示．print() と同義
     #+end_src
   #+reveal: split
   - ベクトルの長さの取得:
     #+begin_src R :tangle yes
       length(x)    
     #+end_src
   - ベクトルの要素の取得:
     #+begin_src R :tangle yes
       x[3] # xの第3要素 (ベクトルの添え字は1から始まる)
       y[c(1,3,4)] # 複数の要素 = c(y[1],y[3],y[4])
     #+end_src
   #+reveal: split
   - 実数 ~a~ から ~b~ まで
     1ずつ変化するベクトル:
     #+begin_src R :tangle yes
       a <- 8; b <- 15 # 変数 a,b に値を代入
       a:b # a < b の場合は1ずつ増加する系列が作成される
       a <- 29.5; b <- 24
       a:b # 逆の場合は1ずつ減少する系列が作成される
       29.5:24 # 直接数値を書いてもよい
     #+end_src
   - 実数 ~a~ から ~b~ まで
     ~c~ ずつ変化するベクトル:
     #+begin_src R :tangle yes
       a <- 1; b <- 16; c <- 2 # 変数 a,b,c に値を代入
       seq(a,b,by=c) # from=a,to=b と明示してもよい    
     #+end_src
   #+reveal: split 
   - ベクトルの反転:
     #+begin_src R :tangle yes
       rev(x)
     #+end_src
   - ベクトルの繰り返し:
     #+begin_src R :tangle yes
       rep(y,3) # 長さは length(y) * 3
       rep(y,times=3) # 単純に繰り返す．上記と同様
       rep(y,each=3)  # 各要素を繰り返す
       rep(y,length.out=12)  # 繰り返した結果の長さを指定する
     #+end_src
   #+reveal: split
   - ベクトルの結合:
     #+begin_src R :tangle yes
       c(x,z) # 同じ属性のもの単純に結合される
       c(x,y) # 異なる属性のものは結合できないので属性が書き換えられる
     #+end_src

** 行列とは
   - スカラー値を2次元状(縦横)に並べたもの
     - 縦(列)ベクトルを列方向に並べて束ねたもの
     - 横(行)ベクトルを行方向に並べて束ねたもの
   - データフレームは行列を拡張したもの

** 行列の生成と操作
   - 基本的には関数 ~matrix()~ を用いて作成する
   - すべての要素が ~a~ である \(m\times n\) 行列の生成:
     #+begin_src R :tangle yes
       a <- 2; m <- 3; n <- 4 # 変数 a,m,n に値を代入
       matrix(a,m,n)
     #+end_src
   #+reveal: split
   - 長さ \(mn\) のベクトル ~a~ を \(m\times n\) 行列に変換:
     #+begin_src R :tangle yes
       a <- 2:13 # 変数 a に値を代入
       (A <- matrix(a,m,n)) # 並び順を変えるには matrix(a,m,n,byrow=TRUE)
     #+end_src
   # - 関数 ~dim()~ によるベクトルから行列への変換:
   #   #+begin_src R :exports code
   #     dim(x) <- c(m,n) # = matrix(x,m,n)
   #   #+end_src
   - 関数 ~as.vector()~ を用いたベクトル化: 
     #+begin_src R :tangle yes
       as.vector(A) # matrix の逆変換にあたる
     #+end_src 
   #+reveal: split
   - 長さが等しい複数のベクトルの結合:
     #+begin_src R :tangle yes
       a <- 4:7; b <- 10:7; c <- c(2,4,8,16) # 変数 a,b,c に値を代入
       rbind(a,b,c) # 行ベクトルとして結合 (row vector)
       cbind(a,b,c) # 列ベクトルとして結合 (column vector)
     #+end_src
     # ~cbind()/rbind()~ は行列の結合にも使える
   #+reveal: split
   - 行列のサイズの取得: 
     #+begin_src R :tangle yes
       dim(A) # 長さ2のベクトル (行数,列数) となることに注意
       nrow(A) # 行数
       ncol(A) # 列数
       dim(A)[1] # nrow(A) と同値
       dim(A)[2] # ncol(A) と同値
     #+end_src
   #+reveal: split
   - 行列の成分の取得:
     #+begin_src R :tangle yes
       A[3,4] # (3,4)成分
       A[3, ] # 第3行のベクトル
       A[ ,4] # 第4列のベクトル
       A[c(1,3),] # 1,3行からなる部分行列
       A[c(1,3),2:4] # 1,3行と，2〜4列からなる部分行列
     #+end_src
     # ベクトルの場合と同様に，
     # 添え字をベクトルで指定することで部分行列の取り出しも可能

** 行列の操作に関する補足
   - 関数 ~cbind()/rbind()~ は行数・列数が等しい行列も横・縦に結合できる
   - 行列の高次元版として配列(array)が用意されている
   - 関数 ~rownames()/colnames()~ を用いると行と列に名前を付けることができる
   - /[[color:gray][これらの機能は講義の中で使いながら説明する]]/

** リストとは
   - 異なる構造のデータを1つにまとめたもの
   - リストの各要素は異なる型であって構わない
   - 本講義のデータ解析ではほとんど用いない
   - Rの関数の操作ではときどき必要

** リストの生成と操作
   - リストの生成:
     #+begin_src R :tangle yes
       (L <- list(x,y)) # x,yを要素とするリスト
     #+end_src
   - リストの要素の参照:
     #+begin_src R :tangle yes
       L[[1]] # リストの第1要素
     #+end_src
   #+reveal: split
   - リストの各要素に名前を付与:
     #+begin_src R :tangle yes
       ## 方法1 (作成時に名前を付与)
       (L1 <- list(first=x, second=y))
     #+end_src
     #+begin_src R :tangle yes
       ## 方法2 (作成後に名前を変更)
       L2 <- list(x,y) 
       names(L2) <- c("first","second")
       L2 # リストを表示
     #+end_src
   #+reveal: split
   - 名前によるリストの要素の取得:
     #+begin_src R :tangle yes
       ## 方法1 (リストの名前で参照)
       L2[["first"]] 
     #+end_src
     #+begin_src R :tangle yes
       ## 方法2 (記号$を用いる場合は""は不要なことに注意)
       L1$first      
     #+end_src

* ベクトルの計算
** ベクトルの表記
   - ベクトルは太字で，要素は下付き添字で表す
     #+begin_src latex
       \begin{equation}
         \boldsymbol{a}
         =(a_{1},a_{2},\dotsc,a_{k})
       \end{equation}
     #+end_src
   - 別の書き方:
     #+begin_src latex
       \begin{equation}
         (\boldsymbol{a})_{i}
         =\text{(ベクトル \(\boldsymbol{a}\) の第 \(i\) 成分)}  
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       a <- c(a1,a2,...,ak) # k次元ベクトルの作成
     #+end_src
   
** ベクトルの加法
   - *同じ長さのベクトル* の和および差: \\
     /[[color:gray][数値の和と差のように扱うことができる]]/
     #+begin_src latex
       \begin{equation}
         \boldsymbol{a}\pm\boldsymbol{b}
         =(a_{1}\pm b_{1},a_{2}\pm b_{2},\dotsc,a_{k}\pm b_{k})
       \end{equation}
       \begin{equation}
         (\boldsymbol{a}\pm\boldsymbol{b})_{i}
         =a_{i}\pm b_{i}
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       a + b # 同じ長さのベクトル a,b の和
       a - b # ベクトルの差
     #+end_src

** ベクトルの乗法
   - *同じ長さの2つのベクトル* の乗法: 
     - 成分ごとの積 (Hadamard積; 要素積)
     - ベクトルの内積
     /[[color:gray][2種類ある]]/
   - データ解析ではどちらも良く用いられる

** Hadamard積 
   - *同じ長さのベクトル* の成分ごとの積
     #+begin_src latex
       \begin{equation}
         \boldsymbol{a}\circ\boldsymbol{b}
         =(a_{1}b_{1},a_{2}b_{2},\dotsc,a_{k}b_{k})
       \end{equation}
       \begin{equation}
         (\boldsymbol{a}\circ\boldsymbol{b})_{i}
         =a_{i}b_{i}
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       a * b # ベクトルの成分ごとの積
       a / b # 成分ごとの商も計算可
     #+end_src
** 内積
   - *同じ長さのベクトル* の内積
     #+begin_src latex
       \begin{align}
         \boldsymbol{a}\cdot\boldsymbol{b}
         &=a_{1}b_{1}+a_{2}b_{2}+\dotsb+a_{k}b_{k}\\
         &=\sum_{i=1}^{k}a_{i}b_{i}
       \end{align}
     #+end_src
   - Rの書式
     #+begin_src R :exports code
       a %*% b # ベクトルの内積
     #+end_src


* 行列の計算
** 行列の表記
   - 行列は大文字で，要素は下付き添字で表す
     #+begin_src latex
       \begin{equation}
         A
         =
         \begin{pmatrix}
           a_{11}&\dots&a_{1n}\\
           \vdots&\ddots&\vdots\\
           a_{m1}&\dots&a_{mn}
         \end{pmatrix}
         % \begin{pmatrix}
         %   a_{11}&a_{12}&\dots&a_{1n}\\
         %   a_{21}&a_{22}&\dots&a_{2n}\\
         %   \vdots&&\ddots&\vdots\\
         %   a_{m1}&a_{m2}&\dots&a_{mn}
         % \end{pmatrix}
       \end{equation}
     #+end_src
   - 別の書き方:
     #+begin_src latex
       \begin{equation}
         (A)_{ij}
         =\text{(行列 \(A\) の \(ij\) 成分)}
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       A <- matrix(c(a11,a21,...,amn),m,n) # m x n 行列の作成
     #+end_src

** 行列の加法
   - *同じ大きさの行列* の和および差: \\
     /[[color:gray][ベクトルと同じように記述することができる]]/
     #+begin_src latex
       \begin{equation}
         (A\pm B)_{ij}=a_{ij}\pm b_{ij}
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       A + B # 同じサイズの行列の和
       A - B # 行列の差
     #+end_src

** 行列の乗法
   - 2つの行列の乗法: 
     - *同じ大きさの行列* の成分ごとの積 (Hadamard積; 要素積)
     - *\(n\times m\) 型行列と \(m\times l\) 型行列* の積
     /[[color:gray][2種類ある]]/
   - データ解析ではどちらも良く用いられる

** Hadamard積
   - *同じ大きさの行列* の成分ごとの積
     #+begin_src latex
       \begin{equation}
         (A\circ B)_{ij}=a_{ij}b_{ij}
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       A * B # 行列の成分ごとの積
       A / B # 成分ごとの商も計算可
     #+end_src
** 行列の積
   - *\(n\times m\) 型行列 \(A\) と \(m\times l\) 型行列 \(B\)* の積
     #+begin_src latex
       \begin{equation}
         (AB)_{ij}=\sum_{k=1}^{m}a_{ik}b_{kj}
         \quad\text{(\(AB\)は\(n\times l\)行列)}
       \end{equation}
     #+end_src
   - Rの書式
     #+begin_src R :eval no
       A %*% B # 行列の積
     #+end_src

** 行列式
   - *\(n\) 次正方行列* \(A\) の行列式
     #+begin_src latex
       \begin{equation}
         \det(A)\quad (=|A|)
       \end{equation}
     #+end_src
     - 正方行列でなければ定義されないことに注意
   - Rの書式
     #+begin_src R :eval no
       det(A) # 行列式
     #+end_src

** トレース
   - *\(n\) 次正方行列* \(A\) のトレース(対角成分の総和)
     #+begin_src latex
       \begin{equation}
         \mathrm{trace}(A)=\sum_{i=1}^{n}a_{ii}
       \end{equation}
     #+end_src
   - Rの書式 (関数は用意されていないので以下を利用):
     #+begin_src R :eval no
       sum(diag(A)) # 行列のトレース
     #+end_src
     - 関数 ~diag()~: 行列の対角成分を取り出す\\
       (ベクトルを対角行列にすることもできる)
     - 関数 ~sum()~: ベクトルの総和を計算する

** 例題
   :PROPERTIES:

   - 適当な2次正方行列 \(A\) で
     Hamilton-Cayleyの定理

       \end{equation}

     ただし \(E_{2}\) は2次単位行列，\(O_{2}\) は2次正方零行列

** 解答例

   :reveal_background: #fef4f4
   :END:
   #+begin_src R :eval no :exports none :tangle yes
     ### 例題 Hamilton-Cayleyの定理の確認
   #+end_src
   #+begin_src R :exports both :results output :tangle yes
     ## 行列を作成 (好きに設定してよい)
     (A <- matrix(1:4,2,2) - diag(rep(3,2)))
     ## 左辺を計算 (丸め誤差の範囲で0になる)
     A %*% A - sum(diag(A)) * A + det(A) * diag(rep(1,2))

** 練習問題
   :PROPERTIES:
   :reveal_background: #fef4f4
   :END:
   - 1から10の2乗値からなるベクトルを作成せよ．
   - 1から10までの和を計算せよ．
   - 行列を用いて九九の表を作成せよ．
   - 30度の回転行列を2回乗ずると60度の回転行列となることを確認せよ．
     #+begin_src latex
       \begin{equation}
         \text{(回転行列)}
         =
         \begin{pmatrix}
           \cos(\theta)&-\sin(\theta)\\
           \sin(\theta)& \cos(\theta)
         \end{pmatrix}
       \end{equation}
     #+end_src

   #+begin_src R :eval no :exports none :tangle yes
     ### 練習問題 ベクトルおよび行列の計算
     ## 1から10までの2乗値からなるベクトル
     1:10 # 1から10までのベクトル
     1:10 * 1:10 # Hadamard積を利用する

     ## 1から10までの和
     1:10 %*% rep(1,10) # (1,2,...,10)と(1,1,...,1)の内積
     sum(1:10) # 関数sum()はベクトルの成分の合計を計算することもできる

** 連立1次方程式の解法
   - 連立1次方程式
     - \(A\) : \(n\) 次正則行列
     - \(\boldsymbol{b},\boldsymbol{x}\) : \(n\) 次元列ベクトル
     #+begin_src latex
       \begin{align}
         A\boldsymbol{x}&=\boldsymbol{b}
         &&\text{(連立1次方程式)}\\
         \boldsymbol{x}&=A^{-1}\boldsymbol{b}
         &&\text{(解;\(A\)が正則な場合)}
       \end{align}
     #+end_src
   - 解を求めるには関数 ~solve()~ を利用する
     #+begin_src R :eval no :tangle yes
       x <- solve(A, b) 
     #+end_src
   - ベクトル \(\boldsymbol{b}\) の代わりに行列も扱える

** 逆行列
   - 正則な \(n\) 次正方行列 \(A\) の逆行列 \(A^{-1}\)
     #+begin_src latex
       \begin{equation}
         AA^{-1}=A^{-1}A=E_{n}
         \quad\text{(\(E_{n}\)は\(n\)次単位行列)}
       \end{equation}
     #+end_src
   - 関数 ~solve()~ を利用して求めることができる
     #+begin_src latex
       \begin{align}
         AX&=E_{n}\\
         X&=A^{-1}E_{n}=A^{-1}
       \end{align}
     #+end_src
     #+begin_src R :eval no
       solve(A,B) # AX=B の解Xを求める
       solve(A) # 逆行列 (Bが単位行列の場合省略できる)
     #+end_src

* COMMENT 制御構造
** 制御文
   - 最適化や数値計算などを行うためには，
     条件分岐や繰り返しを行うための仕組みが必要となる
   - R言語を含む多くの計算機言語では
     - ~if~  (条件分岐)
     - ~for~  (繰り返し・回数指定)
     - ~while~  (繰り返し・条件指定)
   - これを *制御文* と言う 

** ~if~ 文
   - 条件Aが *真* のときプログラムXを実行する
     #+begin_src R :exports code
       if(条件A) プログラムX
     #+end_src
   - 上記の ~if~ 文に条件Aが *偽* のときプログラムYを実行することを追加する
     #+begin_src R :exports code
       if(条件A) プログラムX else プログラムY    
     #+end_src

       fibo <- function(n){
           f0 <- 0 # 第0項の設定
           f1 <- 1 # 第1項の設定
           if(n<0) {
               print("計算できません")
               return(NA) # 欠損値を返す
           }
           if(n==0) { # n=0の場合
               return(f0)
           }
           if(n==1) { # n=1の場合
               return(f1)
           }
           for(i in 2:n) { # n>=2の場合
               fn <- f1 + f0 # fn = fn-1 + fn-2 の計算
               f0 <- f1 # fn-2 の値の更新
               f1 <- fn # fn-1 の値の更新
           }
           return(fn) # 計算結果を返す
       }
